Project mirror is using standard URL syntax to specify protocol and mirrored repository VCS type:
  • git:// - Git (git://git@github.com:Salamek/gitlab-tools.git)
  • https:// - Git over SSL (https://github.com/Salamek/gitlab-tools.git)
  • ssh:// - Git over SSH (ssh://git@github.com/Salamek/gitlab-tools.git)
  • bzr::bzr:// - Bazaar
  • svn+https:// - SVN over SSL
  • svn+ssh:// - SVN over SSH
  • hg::https:// - Mercurial
Other combinations may work, consult GIT documentation.
